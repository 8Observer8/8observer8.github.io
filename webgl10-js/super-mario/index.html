Super Mario clone with free webfussel's sprites

You can run this demo on Android, Windows, and Web (use WAD or arrow keys to move and jump):

  • Click this link to play in the browser
  • Download EXE for Windows 10, 64-bit
  • APK-file for Android


This demo uses free assets only:

  • Free sprites: https://webfussel.itch.io/more-bit-8-bit-mario
  • Free music: https://snabisch.itch.io/mario-pants-music-free
  • Free coin sound: https://opengameart.org/content/picked-coin-echo-2
  • Free jump sound: https://opengameart.org/content/platformer-jumping-sounds

Tools: Qt6, MinGW 64-bit, OpenGL ES 2.0, OpenAL-Soft, Box2D, Free Texture Packer, Tiled Map Editor, Hiero
