Deploy SDL3 app on Android

  • Android SDK, NDK, JDK must be installed
  • Go to this link. Download the "android" archive in the Artifacts section
  • Note. You can find the "INSTALL.md" guide in the "SDL3-3.1.2.aar" archive
  • Create a new empty folder for a new project
  • As the "INSTALL.md" guide said copy "SDL3-3.1.2.aar" to "your-project/libs" folder
  • Create a new text file in your project folder with the "build.gradle" name
