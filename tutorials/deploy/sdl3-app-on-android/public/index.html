Deploy SDL3 app on Android

  • Android SDK, NDK, JDK must be installed
  • Go to this link. Download the "android" archive in the Artifacts section
  • Unzip the downloaded archive somewhere
  • Rename "SDL3-3.1.2.aar" to "SDL3-3.1.2.jar". Unzip the "SDL3-3.1.2.jar" archive
  • You will find the "INSTALL.md" guide in the unzipped "SDL3-3.1.2" folder
  • Create a new empty folder for a new project
  • As the "INSTALL.md" guide said copy "SDL3-3.1.2.aar" to "your-project/libs" folder (so, you should rename it back to ".aar" from ".jar"
  • Create a new text file in your project folder with the "gradle.build" name
